# Clean-Arquitecture-TypeScript
Clean Arquitecture TypeScript Practice


El objetivo es crear un servicio de pedidos sencillo utilizando la arquitectura Clean 

La idea es crear un proyecto que simule el funcionamiento de un monolito con
arquitectura clean code
sera guiada por hitos:
1 dominio modelado solo con entidades y objetos de valor
2 casos de uso que orquesten reglas
3 puertos y adaptadores con persistencia conmutable
4 testing de dominio, aplicacion y contratos
